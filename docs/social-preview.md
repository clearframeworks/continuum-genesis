# Continuum Genesis Social Preview

Use `assets/continuum-genesis-social-preview.png` as the GitHub repository social
preview image.

GitHub path:

```text
https://github.com/clearframeworks/continuum-genesis/settings
```

Steps:

1. Open repository settings.
2. Find `Social preview`.
3. Choose `Edit`.
4. Upload `assets/continuum-genesis-social-preview.png`.
5. Save.
6. Refresh Facebook/Messenger cache with Meta Sharing Debugger:

```text
https://developers.facebook.com/tools/debug/?q=https%3A%2F%2Fgithub.com%2Fclearframeworks%2Fcontinuum-genesis
```

Notes:

- GitHub social preview requires PNG, JPG, or GIF under 1 MB.
- The checked-in PNG is 1280 by 640 and under 1 MB.
- Facebook/Messenger may keep showing the old preview until the URL is scraped again.
